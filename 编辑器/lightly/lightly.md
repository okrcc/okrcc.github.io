# lightly
- 全名`lightly`,后期可能简称`ly`或`light`
- 支持多人协同
- 支持几乎所有主流语言并且不用手动安装, 比如需要编程`java`或`python`不用配置环境
- 支持`windows`和`macos`跨平台
- 今后的语言教学多会用到light和vscode编辑器共同教学
- 安装方便, 使用方便
# 下载
点击[官网下载](https://lightly.teamcode.com/)以进入官网下载或在线编程
# 安装
- 下载好后运行安装包, 打开后选择`我同意`
- 根据需求选择是否为所有用户安装
- 打开安装好的软件, 注册或登录
- 由此, 安装完成
