# input()
- 用法: 接收用户输入
- 返回值为用户输入
使用:
```
input()
输出:
提示: 在输出的地方可以输入, 如:
输出: 这是输入内容
```
打印(输出/print)用户输入的内容
```
print(input())
输入: 123
输出: 123
```
赋值到变量
```
a = input()
```
input可以给出提示语句
```
input('请输入内容:')
输入: 请输出内容: 
```
给出提示, 赋值到变量并输出
```
a = input('请输入:')
print(a)
输出: 请输入: 123
输出: 123
```
